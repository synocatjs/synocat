# PHP Project Template

this is a php project template, you can use it to create your own php project for DSM.

## Project Structure

```bash
$ tree .     
.
├── config
│   └── config.php
├── controllers
│   └── HomeController.php
├── core
│   ├── Controller.php
│   ├── Router.php
│   └── View.php
├── index.php
├── README.md
└── views
    └── home
        └── index.php

```

## Development

```bash
php -S localhost:8080
```
